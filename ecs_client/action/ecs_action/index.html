Traits

ECSRequest

A marker trait to "supertype" each particular kind of ECS request. Each ECSRequest must be serde::ser::Serialize so that it can be serialzied to json and set as the payload (body) of an HTTP request.
ECSResponse

A marker trait to "supertype" each particular kind of ECS response. Each ECSResponse must be serde::de::Deserialize so that it can be deserialized from the json blob in the body of the HTTP response.
